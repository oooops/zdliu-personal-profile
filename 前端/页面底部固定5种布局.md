# 页面底部信息始终固定在底部的5中方法

### 布局方式一：全局增加一个负值下边距等于底部高度
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Title</title>
  <style type="text/css">
    html, body {
      height: 100%;
      margin: 0;
    }
    .warp {
      min-height: 100%;
      margin-bottom: -50px;
    }
    .foot, .push {
      height: 50px;
    }
  </style>
</head>
<body>
<div class="warp">
  <div class="content">content</div>
  <div class="push"></div>
</div>
<div class="foot">foot</div>
</body>
</html>
```

### 布局方式二：底部元素增加负值上边距
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Title</title>
  <style type="text/css">
    html, body {
      height: 100%;
      margin: 0;
    }
    .warp {
      min-height: 100%;
    }
    .content {
      padding-bottom: 50px;
    }
    .foot {
      height: 50px;
      margin-top: -50px;
    }
  </style>
</head>
<body>
<div class="warp">
  <div class="content">content</div>
</div>
<div class="foot">222</div>
</body>
</html>
```


### 布局方式三：使用calc()计算内容的高度
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Title</title>
  <style type="text/css">
    .warp {
      min-height: calc(100vh - 70px);
    }
    .foot {
      height: 50px;
    }
  </style>
</head>
<body>
<div class="warp">content</div>
<div class="foot">foot</div>
</body>
</html>
```

### 布局方式四: 使用flexbox
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Title</title>
  <style type="text/css">
    html, body {
      margin: 0;
    }

    html {
      height: 100%;
    }

    body {
      min-height: 100%;
      display: flex;
      flex-direction: column;
    }

    .warp {
      flex: 1;
    }

    .foot {
      height: 50px;
    }
  </style>
</head>
<body>
<div class="warp">content</div>
<div class="foot">foot</div>
</body>
</html>
```

### 布局方式五: 使用grid布局

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Title</title>
  <style type="text/css">
    html, body {
      margin: 0;
    }

    html {
      height: 100%;
    }

    body {
      min-height: 100%;
      display: grid;
      grid-template-rows: 1fr auto;
    }

    .foot {
      grid-row-start: 2;
      height: 70px;
      grid-row-end: 3;
    }
  </style>
</head>
<body>
<div class="warp">content</div>
<div class="foot">foot</div>
</body>
</html>
```
